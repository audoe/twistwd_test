An extension that adds a "Go To File..." dialog to quickly open (or switch
to) a file in Komodo.

# See Also

- http://community.activestate.com/xpi/quick-open
  Only looks in current project (not cwd, not currently open tabs).
  Has perf problems with large projects.
- "Cmd+T" in TextMate project windows
  Has perf problems with a large base dir.
